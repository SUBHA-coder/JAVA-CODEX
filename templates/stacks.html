Stacks in Java

Understanding Stacks

A stack is a linear data structure that follows the Last-In-First-Out (LIFO) principle. Think of it like a stack of books - you can only add or remove books from the top.

Key Operations:

  • Push - Add an element to the top
  • Pop - Remove the top element
  • Peek - View the top element without removing it
  • isEmpty - Check if stack is empty

Implementation: Array-based Stack


public class Stack {
    private int maxSize;
    private int[] stackArray;
    private int top;
    
    // Constructor
    public Stack(int size) {
        maxSize = size;
        stackArray = new int[maxSize];
        top = -1;
    }
    
    // Push operation
    public void push(int value) {
        if (top < maxSize - 1) {
            stackArray[++top] = value;
            System.out.println(value + " pushed to stack");
        } else {
            System.out.println("Stack is full!");
        }
    }
    
    // Pop operation
    public int pop() {
        if (top >= 0) {
            int value = stackArray[top--];
            System.out.println(value + " popped from stack");
            return value;
        } else {
            System.out.println("Stack is empty!");
            return -1;
        }
    }
    
    // Peek operation
    public int peek() {
        if (top >= 0) {
            return stackArray[top];
        } else {
            System.out.println("Stack is empty!");
            return -1;
        }
    }
    
    // Check if stack is empty
    public boolean isEmpty() {
        return (top == -1);
    }
    
    // Check if stack is full
    public boolean isFull() {
        return (top == maxSize - 1);
    }
}

Usage Example


public class Main {
    public static void main(String[] args) {
        Stack stack = new Stack(3);
        
        // Push elements
        stack.push(10);
        stack.push(20);
        stack.push(30);
        
        // Try to push when stack is full
        stack.push(40); // Will show "Stack is full!"
        
        System.out.println("Top element: " + stack.peek());
        
        // Pop elements
        stack.pop();
        stack.pop();
        stack.pop();
        
        // Try to pop when stack is empty
        stack.pop(); // Will show "Stack is empty!"
    }
}
