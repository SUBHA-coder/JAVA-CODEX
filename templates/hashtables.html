Hash Tables

Theory: Hash tables store key-value pairs for fast lookup. Java uses HashMap for this purpose.

// Java HashMap Example
import java.util.*;
class HashTableDemo {
    public static void main(String[] args) {
        HashMap map = new HashMap<>();
        map.put("apple", 10);
        map.put("banana", 20);
        map.put("orange", 30);
        System.out.println(map.get("banana")); // 20
    }
}

Applications

  • Database indexing
  • Caching
