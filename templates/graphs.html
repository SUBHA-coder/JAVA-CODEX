Graphs

Theory: A graph is a set of nodes (vertices) connected by edges. Can be directed/undirected, weighted/unweighted. Used for modeling networks.

// Java implementation of Graph (Adjacency List)
import java.util.*;
class Graph {
    private int V;
    private LinkedList adj[];
    Graph(int v) {
        V = v;
        adj = new LinkedList[v];
        for (int i = 0; i < v; ++i)
            adj[i] = new LinkedList<>();
    }
    void addEdge(int v, int w) { adj[v].add(w); }
    void printGraph() {
        for (int v = 0; v < V; v++) {
            System.out.print("Vertex " + v + ":");
            for (Integer n : adj[v]) System.out.print(" -> " + n);
            System.out.println();
        }
    }
}

Applications

  • Social networks
  • Navigation systems
