Binary Trees

Theory: A tree is a hierarchical data structure. A binary tree is a tree where each node has at most two children. Common operations include traversals (inorder, preorder, postorder).

// Java implementation of Binary Tree Traversals
class Node {
    int data;
    Node left, right;
    Node(int item) {
        data = item;
        left = right = null;
    }
}
class BinaryTree {
    Node root;
    void inorder(Node node) {
        if (node == null) return;
        inorder(node.left);
        System.out.print(node.data + " ");
        inorder(node.right);
    }
    void preorder(Node node) {
        if (node == null) return;
        System.out.print(node.data + " ");
        preorder(node.left);
        preorder(node.right);
    }
    void postorder(Node node) {
        if (node == null) return;
        postorder(node.left);
        postorder(node.right);
        System.out.print(node.data + " ");
    }
}

Applications

  • Hierarchical data (file systems, organization charts)
  • Expression parsing
